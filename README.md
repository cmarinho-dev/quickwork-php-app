## Quickwork
Este é um MVP criado para demonstrar os conhecimentos adquiridos na disciplina de Desenvolvimento Web em meu curso de Eng. de Software, a ideia é um sistema onde pessoas podem ofertar serviços e outras podem solicitar este serviço ofertado, de modo que seja um processo rápido o qual pode ajudar pessoas em situações financeiras emergenciais;

## Tecnologias
Este projeto é um Projeto Web PHP, que é composto por HTML, CSS, Javascript e Bootstrap, utilizando o PHP para realizar conexão e ter acesso a informações de um banco MySQL.
